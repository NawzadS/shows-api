# Shows API (MVP)

A simple REST API for tracking **TV Shows**, their **Seasons**, and **Episodes** using **Express + Sequelize + SQLite**.

This is the MVP version (no auth yet). It focuses on:
- 3 related resource types (Shows → Seasons → Episodes)
- Full CRUD endpoints
- Proper status codes + error handling
- Basic middleware (JSON parsing + logging)
- Basic unit tests (Jest + Supertest)

---

## Tech Used
- Node.js + Express
- Sequelize ORM
- SQLite
- Jest + Supertest
- dotenv
